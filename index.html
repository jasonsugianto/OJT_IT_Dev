Semi Finalis - Hackathon ElevAIte Indonesia 2025

Mei 2025 - Juni 2025

Membuat aplikasi medis berbasis AI dan IoT berupa dispenser obat. Ada 3 fitur utama aplikasi kami, yaitu user dapat mencatat kapan user minum obat, user dapat menginput foto makanan yang user makan, dan user dapat menambahkan kegiatan lainnya untuk direkam. Tujuan dari aplikasi ini adalah jika suatu saat user sakit, maka dokter juga bisa mengetahui penyebabnya melalui kegiatan-kegiatan yang dilakukannya seperti kedispilinan makan obat serta makanan yang dikonsumsinya.


Peran saya dalam lomba ini adalah membuat dispenser obat yang nantinya akan dihubungkan dengan aplikasinya. Dispenser obat ini dibuat menggunakan tinkercad. Dispenser ini memiliki alarm dan lampu untuk mengingatkan user untuk minum obat. Sebelum menggunakan dispenser ini, user harus input obat yang sedang user konsumsi pada aplikasi.
